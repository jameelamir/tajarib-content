# Reel Caption Format — تجارب

## Version
v1.0 — 2026-02-21

## The Format

```
[Opening sentence: the idea, hook, question, or tension — NOT the guest name. ~10-15 words. Make the viewer stop scrolling.]

[Explaining sentence: what the clip explores + guest name + role woven in naturally. Short.]

تكدرون تشوفون الحلقة الكاملة على منصتكم المفضلة عبر الرابط:
www.tajarib.show 💜
```

## Rules
- Line 1: Always about the IDEA, not the person. Lead with what's at stake.
- Line 2: Guest name + role appear here, framed as their perspective/view — never stated as fact.
- Guest opinions must be attributed: "يرى [الاسم]..." / "بحسب [الاسم]..." / "يقول [الاسم]..."
- CTA is always the same — don't vary it.
- No hashtags unless explicitly asked.
- No command forms (شوفوا، اسمعوا) — use invitational: تكدرون.
- Language: Iraqi white language (لغة بيضاء) — not fusha, not heavy colloquial.
- Don't use چ → use ج. Say وية not ويا.
- Avoid يحجي → use نتكلم، نناقش، نغطي instead.
- Length: 2-4 lines total (excluding CTA). Short. Punchy.
